# Requirements
- Visual Studio.
- .NET MAUI packages from Visual Studio Installer.
- .NET community Toolkit
- Android Emulator[Can choose from any version from Android 5.0 to 14.0 but works best with Android 9.0]. Systems with better PC video cards are recommended for running this application Or 
  else Hardware Acceleration need to be activated. To know more (https://learn.microsoft.com/en-us/dotnet/maui/android/emulator/hardware-acceleration?view=net-maui-8.0#accelerate-with-aehd) 
- For MAC and IOS the respective devices are needed for authorizing and deployment.(Since I dont have a MacBook its not tested in IOS and MAC platforms but it will work and you can try :) ).
